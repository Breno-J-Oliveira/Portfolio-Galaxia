# Assets — Imagens necessárias

Substitua os arquivos placeholder abaixo com seus arquivos reais.

## Obrigatórios

| Arquivo         | Dimensão     | Formato | Descrição                           |
|-----------------|--------------|---------|-------------------------------------|
| `foto.jpg`      | 400×400px+   | JPG     | Foto de perfil (quadrada)           |
| `og-image.jpg`  | 1200×630px   | JPG     | Imagem Open Graph (redes sociais)   |
| `curriculo.pdf` | —            | PDF     | Currículo para download             |

## Opcionais

| Arquivo / Pasta  | Descrição                                      |
|------------------|------------------------------------------------|
| `fotos/`         | Fotos dos clientes para os depoimentos (80×80px JPG) |
| `favicon.ico`    | Ícone do site na aba do navegador (32×32 ICO) |
| `favicon.svg`    | Versão SVG do favicon (moderno)               |

## Dicas

- **foto.jpg**: Use uma foto profissional com fundo neutro ou escuro para combinar com o tema.
- **og-image.jpg**: Crie em [Canva](https://canva.com) — template 1200×630px. Inclua seu nome, cargo e uma cor de destaque.
- **curriculo.pdf**: Versão PDF do seu currículo. Recomendado: 1-2 páginas, formato ATS-friendly.
- Comprima as imagens em [TinyPNG](https://tinypng.com) antes de subir para o servidor.
