# Áudios da seção Poliglota

Coloque aqui os áudios de você falando cada idioma. Os nomes devem bater com `data/index.js` → `polyglot.languages[].audio`:

- `pt.mp3` — Português
- `en.mp3` — English
- `es.mp3` — Español
- `fr.mp3` — Français

Enquanto os arquivos não existirem, os players aparecem na página mas não tocam.
Depois de adicionar, inclua os caminhos no array `ASSETS` do `sw.js` e faça o bump do cache.
